    Exude API

    Version: 1.0.0

    Open API Swagger Spec

    Terms of service Terms

    /exude/stopping/data

    POST

    Summary

    Filter the stopping words from the provided input data or links

    Parameters
    Name Located in Description Required Schema
    type path provide the type of filtering required stopping Yes string
    Request Body

    Pass the ExudeRequestBean Object, pass either text data or data links

    { "data": "string", "links": [ "string" ] }

    Responses
    Code Description
    200 Exude response

    Sample

    RequestBody: { "links":[ "https://en.wikipedia.org/wiki/Rama" ] }

    ResponseBody: { "data": { "https://en.wikipedia.org/wiki/Rama": "ramakien 516832 half tdr upload latina idol sivananda kathakali .... } }
